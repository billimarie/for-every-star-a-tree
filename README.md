# Plant Research for the Antelope Valley Climate
> Instructions on how to help us with our open-source project

Since I am not a plant expert, I have been utilizing different tools and websites for my research, then verifying with local plant experts who are growing in our desert region.

I'm making this list an open-source resource so that other desert garden projects in the Antelope Valley can find it.

Maybe together we can heal degraded land by cultivating greener spaces.

## How you can help

1. Go to this Calscape.org list: [Lake Los Angeles native plants](https://calscape.org/loc-34.6125,-117.8281-(M6R2-C4-Lake-Los-Angeles,-CA,-USA)/cat-All-Plants/ord-popular/page-1/np-0?srchcr=sc64db833cab014). I'm not sure how often this is updated or exactly how accurate it is, but it is considered an established source of truth by various plant professionals in SoCal.

2. Pick a plant.

3. Google the content info of a random nursery in the area.
  
5. Ask if they have the plant in stock. If yes, please write down the Size and Price.

6. Add the plant to the list.

Having a single source of plants is a good start to organizing the desert forest project.

Thanks for helping us grow.
